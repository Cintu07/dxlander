### Key Commands

```bash
# Get started quickly
git clone https://github.com/dxlander/dxlander.git
cd dxlander
pnpm install
pnpm dev

# Development workflow
pnpm build        # Build all packages
pnpm test         # Run tests
pnpm lint         # Check code style
pnpm typecheck    # Verify types
```
